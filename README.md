# Mini-Project-on-Financial-Analysis
Conducted analysis of financial sample data to identify sales variations and highlight growth trends.
Cleaned and prepared a 1000+ row dataset by correcting removing duplicates, missing values and resolving formatting issues using Excel and Power Query Editor.
Applied DAX measures to calculate key metrics (total sales, orders, sales from last year, profit, profit margin, discounts) and enhanced dashboards with cards, filters, and slicers.
Built interactive dashboards in Power BI to visualize trends and support decision-making.
